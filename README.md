# 🍪 Biscoito da Sorte - React Native

Um aplicativo simples e divertido que exibe frases irônicas e cínicas no estilo "biscoito da sorte".

## Requisitos

- Node.js (>= 18 recomendado)
- npm ou yarn
- Expo CLI

## Executando

Clone o repositório

```bash
git clone https://github.com/seu-usuario/biscoito_sorte.git
cd biscoito_sorte
```
Instale as dependências e execute
```bash
npm install
npx expo start
```
Pressione *w* para abrir na web ou escaneie o QRcode para abrir no celular, caso já tenha o aplicativo Expo Go instalado.
